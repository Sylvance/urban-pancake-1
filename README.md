## Installation
Clone the repo;
`git clone https://github.com/Sylvance/urban-pancake-1.git`

Change directory;
`cd urban-pancake-1`

Install node modules;
`npm install`

Start server;
`node app.js`

Change directory in another terminal;
`cd urban-pancake-1`
`cd socket-io-client`

Install node modules;
`npm install`

Start react server;
`npm start`
